# Python-Adaptive-Upscaler
 - Upscales &amp; Enhances Videos, Gifs, and Images to the size you want

Run the "Requirements" file to install Python Requirements

This also needs FFMPEG
FFMPEG Installer can be found here

This also needs Python
Python can be found here: https://www.python.org
Python to PATH Install guide: https://youtu.be/H7iqfv5dzRY

Put your media (Videos Only) in the "MEDIA" folder
Run the "Start" file
Pick a resolution
Let the script run
Your upscaled/downscaled media is in the "OUTPUT" folder

DOES NOT ENHANCE QUALITY BECAUSE IT DOES NOT USE AI
